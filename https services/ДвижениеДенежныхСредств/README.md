# Движение денежных средств (GetCashFlows)

HTTP-сервис возвращает ведомость по денежным средствам: остатки и обороты по счетам и кассам из регистра накопления **ДенежныеСредства**.

**HTTP-сервис:** `cashflows`  
**Метод:** `get`  
**Функция в конфигураторе:** `GetCashFlows`

Ответ всегда имеет формат:

```json
{
  "data": [],
  "errors": []
}
```

## Endpoint

```http
GET /main/hs/cashflows/get
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `from` | дата | Начало периода. Поддерживаются `dd.MM.yyyy`, `dd-MM-yyyy`, `yyyy-MM-dd`. Если не указан, период начинается с начала учета. |
| `to` | дата | Конец периода. Если не указан, используется конец текущего дня сеанса 1С. |
| `date` | дата | Один день среза. Имеет приоритет над `from` и `to`. |
| `org` | строка | Подстрока в организации. |
| `account` | строка | Подстрока в счете или кассе. |
| `cash` | строка | Подстрока в счете или кассе, с тем же смыслом, что `account`. |
| `bank` | строка | Подстрока в банке. |
| `kind` | строка | Вид денежных средств: наличные, безналичные или подстрока в представлении. |
| `accountType` | строка | Подстрока в виде счета. |
| `currency` | строка | Подстрока в валюте. |
| `closed` | булево | Фильтр по признаку закрытого счета. Истина: `1`, `true`, `yes`, `да`. |
| `mgmt` | булево | Использовать управленческие суммы регистра. |
| `totals` | булево | Добавить итоговую строку по выбранным данным. |

## Поля ответа

Каждая строка `data` содержит:

| Поле | Описание |
|------|----------|
| `Организация` | Представление организации. |
| `СчетКасса` | Счет или касса. |
| `ВидДенежныхСредств` | Наличные или безналичные. |
| `Банк` | Банк счета, если применимо. |
| `ВидСчета` | Вид банковского счета, если применимо. |
| `Валюта` | Валюта счета или кассы. |
| `Закрыт` | Признак закрытого счета. |
| `ПериодС` | Начало периода ответа. |
| `ПериодПо` | Конец периода ответа. |
| `НачальныйОстаток` | Остаток на начало периода. |
| `Приход` | Приход за период. |
| `Расход` | Расход за период. |
| `КонечныйОстаток` | Остаток на конец периода. |

При `totals=true` в конец массива добавляется строка с `ЭтоИтог = true`.

## Примеры

```http
GET /main/hs/cashflows/get
GET /main/hs/cashflows/get?from=01.02.2025&to=28.02.2025
GET /main/hs/cashflows/get?date=15.02.2025
GET /main/hs/cashflows/get?org=Демо&bank=СБЕР&kind=Безналичные
GET /main/hs/cashflows/get?from=01.02.2025&to=28.02.2025&mgmt=true&totals=true
```
