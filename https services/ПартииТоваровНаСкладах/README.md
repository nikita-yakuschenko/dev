# Стоимость остатков по складам (GetStockBalances)

HTTP-сервис возвращает суммарную стоимость остатков по складам. Используется отдельный сервис, чтобы не конфликтовать с существующим серверным модулем `balances/GetBalances` из набора `pulse`.

**HTTP-сервис:** `stockbalances`  
**Метод:** `get`  
**Функция в конфигураторе:** `GetStockBalances`

В публикации 1С значение **Корневой URL** должно быть ровно `stockbalances`.
Если в публикации указан `stockbalaces`, запрос к `/hs/stockbalances/get`
вернет IIS 404, потому что опубликован другой URL.

Источник данных: **РегистрНакопления.ПартииТоваровНаСкладах.Остатки**.

Ответ всегда имеет формат:

```json
{
  "data": [],
  "errors": []
}
```

## Endpoint

```http
GET /<публикация>/hs/stockbalances/get
```

Например для тестовой публикации `module.team`:

```http
GET /module.team/hs/stockbalances/get?date=01.07.2026
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `warehouse` | строка | Фильтр по наименованию склада. |
| `date` | дата | Дата среза остатков. Поддерживаются `dd.MM.yyyy`, `dd-MM-yyyy`, `yyyy-MM-dd`. Если не указан, используется конец текущего дня сеанса 1С. |
| `code` | строка | Опциональный фильтр по коду номенклатуры перед суммированием. Поиск `ПОДОБНО`. |
| `name` | строка | Опциональный фильтр по наименованию номенклатуры перед суммированием. Символ `/` в query передается как `_slash_`. |

## Поля ответа

Каждая позиция `data` содержит:

| Поле | Описание |
|------|----------|
| `Склад` | Наименование склада. |
| `ДатаСреза` | Дата среза в формате `dd.MM.yyyy`. |
| `Стоимость` | Суммарная стоимость остатков на складе. |

## Примеры

```http
GET /<публикация>/hs/stockbalances/get
GET /<публикация>/hs/stockbalances/get?date=01.07.2026
GET /<публикация>/hs/stockbalances/get?warehouse=Северный
GET /<публикация>/hs/stockbalances/get?name=бетон
GET /<публикация>/hs/stockbalances/get?code=ДК-00001008
```

## Отличие от balances

`balances/GetBalances` возвращает количественные остатки по регистру **ТоварыНаСкладах**.  
`stockbalances/GetStockBalances` возвращает суммарную стоимость остатков по складам из регистра **ПартииТоваровНаСкладах**.
