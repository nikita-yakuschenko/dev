# Стоимость остатков товаров (GetStockBalances)

HTTP-сервис возвращает количественные и стоимостные остатки номенклатуры по складам. Используется отдельный сервис, чтобы не конфликтовать с существующим серверным модулем `balances/GetBalances` из набора `pulse`.

**HTTP-сервис:** `stockbalances`  
**Метод:** `get`  
**Функция в конфигураторе:** `GetStockBalances`

Источник данных: **РегистрНакопления.ПартииТоваровНаСкладах.Остатки**.

Ответ всегда имеет формат:

```json
{
  "data": [],
  "errors": []
}
```

## Endpoint

```http
GET /main/hs/stockbalances/get
```

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `code` | строка | Фильтр по коду номенклатуры. Поиск `ПОДОБНО`. |
| `name` | строка | Фильтр по наименованию номенклатуры. Символ `/` в query передается как `_slash_`. |
| `warehouse` | строка | Фильтр по наименованию склада. |
| `date` | дата | Дата среза остатков. Поддерживаются `dd.MM.yyyy`, `dd-MM-yyyy`, `yyyy-MM-dd`. Если не указан, используется конец текущего дня сеанса 1С. |
| `full` | булево | При `true` в ответ добавляется вся отфильтрованная номенклатура, включая позиции без строк остатков. Истина: `1`, `true`, `yes`, `да`. |

## Поля ответа

Каждая позиция `data` содержит:

| Поле | Описание |
|------|----------|
| `Код` | Код номенклатуры. |
| `Наименование` | Наименование номенклатуры. |
| `ЕдиницаИзмерения` | Базовая единица измерения. |
| `ВидНоменклатуры` | Вид номенклатуры. |
| `НоменклатурнаяГруппа` | Номенклатурная группа. |
| `ДатаСреза` | Дата среза в формате `dd.MM.yyyy`. |
| `ИтогоКоличество` | Суммарное количество по складам. |
| `ИтогоСтоимость` | Суммарная стоимость по складам. |
| `СредняяЦена` | Средняя цена по позиции: `ИтогоСтоимость / ИтогоКоличество`, округление до 4 знаков. |
| `Остатки` | Массив остатков по складам. |

Элемент массива `Остатки`:

| Поле | Описание |
|------|----------|
| `Склад` | Наименование склада. |
| `Количество` | Количество на складе. |
| `Стоимость` | Стоимость остатка на складе. |
| `СредняяЦена` | Средняя цена на складе: `Стоимость / Количество`, округление до 4 знаков. |

## Примеры

```http
GET /main/hs/stockbalances/get
GET /main/hs/stockbalances/get?name=бетон
GET /main/hs/stockbalances/get?code=ДК-00001008
GET /main/hs/stockbalances/get?warehouse=Северный
GET /main/hs/stockbalances/get?date=15.03.2025
GET /main/hs/stockbalances/get?name=кирпич&full=true
```

## Отличие от balances

`balances/GetBalances` возвращает количественные остатки по регистру **ТоварыНаСкладах**.  
`stockbalances/GetStockBalances` возвращает стоимость остатков по партиям товаров.
